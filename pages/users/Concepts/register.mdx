# Register your STARK domain

The Starknet.ID application enables users to register STARK domain names within the Starknet network. Once registered, these STARK domain become usable across all applications that integrate Starknet.ID. This seamless integration allows users to employ their registered domain names across a spectrum of Starknet applications.

****Get Started Now****

To register a STARK domain, you need to:

1/ Navigate to the Starknet.ID application https://app.starknet.id/

2/ Connect your wallet

3/ Choose your STARK domain

![Choose domain.png](https://prod-files-secure.s3.us-west-2.amazonaws.com/b7f7f283-89cc-461e-8346-69f3510d0d81/f6669410-9529-4859-81f9-bcdbfd8b5518/Choose_domain.png)
